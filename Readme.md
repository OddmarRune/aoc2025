## Advent of Code 2025

Only test problems included. The actual problem input should be in the form `input/input01.txt`.

### Building
```bash
$ make
```

### To run a test:
```bash
$ bin/day1_part1 test
```

### Solve a problem
```bash
$ # Remember to put the actual problem in input/input01.txt
$ bin/day1_part1
```

### To run all tests
```bash
$ ./run.sh test
```

### To solve all problems
```bash
$ ./run.sh
```
