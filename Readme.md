## Advent of Code 2025

Only test problems included. The actual problem input should be in the form `input/input01.txt`.

### Building
```bash
$ make
```

### To run a test:
```bash
$ bin/day1_part1 test
```

### Solve a problem
Remember to put the actual problem in `input/input01.txt`.
```bash
$ bin/day1_part1
```

### To run one test
Both part 1 and 2.
```bash
$ ./run.sh 1 test 
# or
$ ./run.sh test 2
```

### To run all tests
```bash
$ ./run.sh test
```

### To solve one problem
```bash
$ ./run.sh 5
```

### To solve all problems
```bash
$ ./run.sh
```
